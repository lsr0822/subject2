## top, ps, jobs, kill 명령어 정리

### 1. top
-실시간으로 시스템의 프로세스, CPU/메모리 사용량을 모니터링하는 명령어

1. 주요 기능
   - CPU, 메모리 사용률을 실시간으로 표시
   - 프로세스 목록을 자원 사용량 순으로 정렬
   - 시스템 부하, 실행 중인 프로세스 수 등의 정보를 제공
  
     
2. 사용법
   - `top`

   
3. 주요 단축키
   - `q` : 종료
   - `k` : 특정프로세스 종료
   - `M` : 메모리 사용량 기준 정렬
   - `P` : CPU 사용량 기준 정렬
  
### 2. ps
   -현재 실행 중인 프로세스를 스냅샷 형태로 출력하는 명령어

1. 주요 옵션
   - `ps aux` : 모든 프로세스를 자세히 표시
   - `ps -ef` : 모든 프로세스를 full format으로 표시
   - `ps -U 사용자명` : 특정 사용자의 프로세스만 표시

2. 출력 정보
   | 항목 | 설명 |
   | ---- | ---- |
   | PID | 프로세스 ID |
   | USER | 프로세스 소유자 |
   | %CPU | CPU 사용률 |
   | %MEM | 메모리 사용률 |
   | COMMAND | 실행 명령어 |
  
### 3. jobs
    -현재 쉘에서 백그라운드 작업을 확인하는 명령어(현재 터미널에서 실행한 작업만 확인 가능)

1. 사용법
   - `jobs` : 작업 목록 표시
   - `jobs -l` : PID와 함께 표시

2. 작업 상태
   | 항목 | 설명|
   | ---- | ---- |
   | Running | 실행중 |
   | Stopped | 일시 중지됨 |
   | Done | 완료됨 |

### 4. kill
: 프로세스에 시그널을 보내 종료하거나 제어하는 명령어이다.

1. 사용법
   - `kill PID` : 프로세스 정상 종료 요청
   - `kill -9 PID` : 프로세스 강제 종료
   - `kill -15 PID` : 정상 종료

2. 주요 시그널
   - `SIGTERM (15)` : 정상 종료 요청 (기본값)
   - `SIGKILL (9)` : 강제 종료
   - `SIGHUP (1)` : 재시작 (설정 파일 다시 읽기)
   - `SIGSTOP (19)` : 일시 정지
