## top, ps, jobs, kill 명령어 정리

⸻

# 1. top

실시간으로 시스템의 CPU·메모리 사용량과 현재 동작 중인 프로세스를 모니터링하는 명령어.

<b>주요 기능</b>
-CPU·메모리 사용률을 실시간으로 확인
-자원 사용량 기준으로 프로세스 정렬
-시스템 부하, 실행 중 프로세스 수 등 전체 시스템 상태 파악 가능

<b>사용 방법</b>

top

<b>주요 단축키</b>
-q : 종료
-k : 특정 프로세스 종료 요청
-M : 메모리 사용량 기준 정렬
-P : CPU 사용량 기준 정렬

⸻

# 2. ps

현재 실행 중인 프로세스를 스냅샷 형태로 출력하는 명령어.

<b>주요 옵션</b>
-ps aux : 모든 프로세스를 상세하게 표시
-ps -ef : Full format으로 출력
-ps -U 사용자명 : 특정 사용자의 프로세스만 표시

<b>출력 항목 설명</b>

<b>항목	의미</b>
PID	프로세스 ID
USER	소유자
%CPU	CPU 사용률
%MEM	메모리 사용률
COMMAND	프로세스를 실행한 명령어



⸻

# 3. jobs

현재 현재 쉘에서 실행한 백그라운드 작업 목록을 확인하는 명령어.
(다른 터미널에서 실행한 프로세스는 표시되지 않음)

<b>사용 방법</b>
-jobs : 백그라운드 작업 확인
-jobs -l : PID 포함하여 표시

<b>작업 상태 종류<b/>

 상태	설명
Running	실행 중
Stopped	일시 정지
Done	작업 완료



⸻

# 4. kill

프로세스에 시그널을 보내 종료하거나 제어하는 명령어.

<b>사용 방법</b>
-kill PID : 정상 종료 요청
-kill -9 PID : 강제 종료
-kill -15 PID : 정상 종료 요청(Sigterm)

<b>주요 시그널</b>
-SIGTERM (15) : 정상 종료 요청(기본)
-SIGKILL (9) : 강제 종료
-SIGHUP (1) : 재시작(설정 파일 재로딩)
-SIGSTOP (19) : 프로세스 일시 정지
